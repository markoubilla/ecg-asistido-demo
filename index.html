Prototipo funcional navegable

Demo para revision clinica y tecnica. Recorre el flujo de una lectura ECG desde captura hasta registro y confirmacion intrahospitalaria.

Uso previsto de esta demo

Este prototipo no analiza ECG reales ni entrega diagnosticos. Simula el flujo, los mensajes de seguridad y la experiencia de usuario para recoger opinion de medicos, equipos prehospitalarios y referentes tecnicos.

Flujo MVP

Simulacion

Puedes alternar entre una captura interpretable y una no interpretable para probar los mensajes de seguridad.

Salida esperada

El prototipo guarda casos solo en memoria del navegador. Sirve para validar experiencia, lenguaje clinico, seguridad y utilidad antes de pasar a Flutter, React Native o una prueba piloto controlada.
